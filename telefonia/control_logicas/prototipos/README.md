# Laboratorio visual LCM

Comparador estático de tres direcciones visuales para la futura unificación de La Cablera Marplatense.

- Bruma: suave, serena y operativa.
- Atlántica: técnica, compacta y precisa.
- Lino: cálida, sobria y editorial.

No consulta APIs, no usa credenciales y no modifica el módulo operativo. Los datos visibles son ejemplos estáticos.

Abrir con el servidor PHP local:

```text
http://127.0.0.1:8091/telefonia/control_logicas/prototipos/
```

Tambien se puede abrir una variante directa:

```text
?tema=bruma
?tema=atlantica
?tema=lino
```
